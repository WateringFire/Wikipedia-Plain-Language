Tips

  • When contributing to a Simple Wikipedia article, use this list to check. This is not a list of all requirements your article should have. This list is meant to check minimum requirements:
    • Use simple words. Define complex terms if you need to include them.
    • Use positive language. Avoid writing double negatives.
    • Use short paragraphs.
    • Use short sentences. Avoid compound sentences.
    • Avoid technical and unnecessary jargon.
    • Use active voice.
    • Use present tense.
    • Use examples and definitions to explain bigger ideas.
    • Use headers.
    • Use transition words (e.g., “for example,” “also,” “next”).

FAQ

  • Q: This is my first time contributing to Wikipedia. How can I start contributing?
  • A: Anyone can contribute to Simple Wikipedia as a guest. However, make an account if you want to protect your privacy. You do not need a Wikipedia account to access Simple Wikipedia articles.
  • Q: How do I search for Simple Wikipedia articles?
  • A: Use the search bar to search if a Simple Wikipedia article translation exists.
